# SOIP

Student organization information project 学生组织信息化项目

## 内容

学生组织信息化项目是长春大学计算机科学技术学院学生会发起的学生组织标准化实践项目之一。

为贯彻落实中青联发【2023】3号关于巩固高校学生会（研究生会）改革成果的若干措施，特提出本项目。

## 项目列表
